Sistema Comercial (PDV Desktop)

Aplicação de **Ponto de Venda (PDV)** desenvolvida em **C# (Windows Forms)** para gestão de vendas em ambiente comercial.  
O sistema foi projetado para oferecer uma solução simples e eficiente para controle de operações de venda em estabelecimentos de pequeno e médio porte.

---

Estrutura do Projeto

- **DAL/** → Camada de acesso a dados (Data Access Layer)  
- **Database/** → Scripts e arquivos relacionados ao banco de dados  
- **Forms/** → Interfaces gráficas (Windows Forms)  
- **Utils/** → Funções auxiliares e utilitários  

---

Funcionalidades

- Cadastro e gerenciamento de produtos  
- Registro de vendas com interface amigável  
- Integração com banco de dados para persistência  
- Relatórios gráficos de movimentação comercial  
- Estrutura modular para futuras expansões  

---

Tecnologias Utilizadas

- **Linguagem:** C#  
- **Framework:** .NET (Windows Forms)  
- **Banco de Dados:** MySQL / SQL Server (dependendo da configuração)
